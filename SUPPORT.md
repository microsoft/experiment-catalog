# Support

This repository is maintained by Microsoft and is under active development.

Support for Microsoft products and services used by this repository does not extend to this repository. The project is provided under the repository license and is supported through the collaboration channels in this repository.

## How to get help

* Search existing issues before opening a new report.
* Use repository issues for bug reports, feature requests, and usage questions.
* Use [SECURITY.md](./SECURITY.md) for suspected vulnerabilities instead of public issues.

## Response expectations

* We review issues and pull requests as capacity allows.
* Response times vary and no service-level agreement applies to this repository.
* If the support model changes, this file will be updated to reflect the new expectation.

## Contributing

* Review [CONTRIBUTING.md](./CONTRIBUTING.md) before submitting a pull request.
